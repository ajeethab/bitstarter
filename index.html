The content tracker toolbox is envisaged as a cloud-based SaaS application that will help users to track changes to content on third party websites. Typically this can be used to track changes to laws, regulations and similar content that is posted on certain govermnment websites such as SEC for example.The application can also be used to check for changes to content that is updated periodically, or e.g., weekly or monthly updates for content. This application will allow the user to define his / her set of tracked content sets and provide the user the functionality to define rules to identify and track changes, maintain a repository of changes in the application, and the ability to export the changes to a defined XML / CSV format.

In future the application will also support ability to mashup data from multiple sources based on defined rules, ability to export to PDF and other functionalities. The initial version is envisaged as a trial version which will allow each user to setup up to a maximum of three content sources to be tracked along with associated rules.
